# chatroom-server

## Features
 - Use node.js
 - Use docker

## Install
```bash
git clone <reop>
```

## Commands
 - Before development
```
npm install
```

 - Start server
```
node server.js
```

## Docker
 - Build docker image
 ```
 sh docker-build-backend.sh
 ```

 - Run image
 ```
 docker-compose up -d
 ```
